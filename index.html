代表项目 · 能力证据

wechat-pc-remote

一个用微信远程控制 Windows 电脑的 AI Coding 项目。它证明我能把手机入口、Agent、Skill、PowerShell 脚本和本地自动化串成真实可用的系统。

1 微信指令
帮我看一下桌面，并总结今天的待办事项
打开 C:\Projects 文件夹
查找过去 7 天的销售数据，生成趋势图
把这份数据整理成表格并导出 Excel
写一份今日工作总结
2 Agent 推理
  1. 理解任务 识别意图：查看桌面并总结待办
  2. 选择技能 调用截图、OCR、文件和报告相关 Skill
  3. 执行动作 PowerShell 固定脚本完成本地操作
  4. 生成回复 把结果整理成用户能直接看的中文回复
3 Windows 输出
  • 桌面包含项目评审材料、销售数据和今日总结草稿。
  • 最新文件：sales_trend_7days.xlsx。
  • 需要跟进：团队周会 15:00，报告初稿校对。
  • 建议：先整理数据，再输出一页结论。

今日工作总结

完成桌面信息读取、数据趋势整理和报告草稿生成。
